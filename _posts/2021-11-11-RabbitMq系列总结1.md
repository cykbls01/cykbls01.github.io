# RabbitMq系列总结1

Docker环境安装RabbitMq
--

RabbitMq的安装很简单，只需要启动一个容器即可。

---

## 安装RabbitMq

RabbitMq的安装需要暴露出15672和5672端口供外界使用，一个是web访问，另一个是api端口。

```bash
docker run --name rabbitmq -d -p 15672:15672 -p 5672:5672 rabbitmq:3.8-management
```
