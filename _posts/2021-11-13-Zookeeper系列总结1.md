# Zookeeper系列总结1

Docker环境安装Zookeeper
--

Kafka的安装总共分为三个部分，分别是zookeeper，kafka以及kafka-manager的安装。

---

## 安装Zookeeper

zookeeper的安装需要暴露出2181端口供外界使用，

```bash
docker run -d --name zookeeper --publish 2181:2181 --volume /etc/localtime:/etc/localtime --restart=always wurstmeister/zookeeper
```
