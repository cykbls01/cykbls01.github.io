# MongoDB系列总结1

Docker环境安装MongoDB
--

MongoDB的安装非常简单，只需要启动一个容器即可。

---

## 安装MongoDB

MongoDB安装需要暴露27017端口，同时可以直接使用以下命令进入容器。

```bash
docker run -itd --name mongo -p 27017:27017 mongo
#使用该命令进入容器
docker exec -it mongo mongo admin
```
