# Redis系列总结1

Docker环境安装Redis
--

Redis的安装非常简单，只需要启动一个容器即可。

---

## 安装Redis

Redis安装需要暴露6379端口，同时可以直接使用以下命令进入容器。

```bash
docker run -itd --name redis-test -p 6379:6379 redis
#使用该命令进入容器
docker exec -it redis-test /bin/bash
```
